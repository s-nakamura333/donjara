# ドンジャラ得点計算アプリ

「ドンジャラNEO ゴジラ」対応の得点計算アプリです。  
OCRとChatGPTを組み合わせて、初心者や子供でも得点計算が簡単にできるよう設計しました。

## 🔧 使用技術
- Kotlin
- Android Studio
- Google ML Kit（OCR）
